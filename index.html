DNS Pro hizmetini kullanarak aşağıdaki şartları kabul etmiş sayılırsınız:

  • Hizmetimizi kötüye kullanmayacaksınız
  • Oluşturulan scriptleri yalnızca kendi cihazlarınızda kullanacaksınız

2. Sorumluluk Reddi

DNS Pro olarak:

  • DNS değişikliğinin yasal sonuçlarından sorumlu değiliz
  • Hizmet kesintilerinden veya bağlantı sorunlarından sorumlu değiliz
  • Üçüncü parti DNS sağlayıcılarının performansından sorumlu değiliz
  • Oluşturulan scriptlerin kullanımından doğacak sorunlardan sorumlu değiliz

3. Hizmet Değişiklikleri

Politikalarımızı ve hizmetlerimizi önceden haber vermeksizin değiştirme hakkını saklı tutarız.

Gizlilik Sözleşmesi

DNS Pro olarak şu verileri toplayabiliriz:

  • Seçtiğiniz DNS sağlayıcı bilgisi (anonim olarak)

Genel Sorular

DNS nedir ve ne işe yarar?

DNS (Domain Name System), internet adreslerini IP adreslerine çeviren sistemdir. Tarayıcınıza yazdığınız web adreslerinin hangi sunucuya bağlanacağını belirler.

Neden DNS değiştirmeliyim?

DNS değiştirmek internet hızınızı artırabilir, bazı engelli sitelere erişim sağlayabilir veya güvenliğinizi iyileştirebilir.

Hangi DNS sağlayıcı en hızlı?

Genellikle Google DNS ve Cloudflare en hızlı seçeneklerdir, ancak konumunuza göre değişiklik gösterebilir.

Teknik Sorular

DNS değiştirmek yasal mı?

DNS değiştirmek teknik olarak yasaldır, ancak yerel yasalarınıza göre bazı sitelere erişim yasal olmayabilir.

Script çalıştırmadan DNS değiştirebilir miyim?

Evet, ağ ayarlarınızdan manuel olarak da DNS değiştirebilirsiniz ancak bu işlem işletim sistemine göre değişir.
