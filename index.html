Kullanım Politikaları

1. Kullanım Şartları

DNS Pro hizmetini kullanarak aşağıdaki şartları kabul etmiş sayılırsınız:

  • Bu araç yalnızca yasal amaçlar için kullanılacaktır
  • Yerel yasalarınıza uygun hareket etmek sizin sorumluluğunuzdadır
  • Hizmetimizi kötüye kullanmayacaksınız

2. Sorumluluk Reddi

DNS Pro olarak:

  • DNS değişikliğinin yasal sonuçlarından sorumlu değiliz
  • Hizmet kesintilerinden veya bağlantı sorunlarından sorumlu değiliz
  • Üçüncü parti DNS sağlayıcılarının performansından sorumlu değiliz

3. Hizmet Değişiklikleri

Politikalarımızı ve hizmetlerimizi önceden haber vermeksizin değiştirme hakkını saklı tutarız.

Gizlilik Sözleşmesi

1. Toplanan Veriler

DNS Pro olarak şu verileri toplayabiliriz:

  • Seçtiğiniz DNS sağlayıcı bilgisi (anonim olarak)
  • Temel kullanım istatistikleri
  • Tarayıcı bilgileri ve IP adresi (anonimleştirilmiş)

2. Veri Kullanımı

Topladığımız verileri:

  • Hizmetimizi geliştirmek için kullanırız
  • Reklam performansını ölçmek için kullanırız
  • Asla üçüncü şahıslara satmayız

3. Çerezler

Web sitemiz temel işlevsellik için çerezler kullanır. Çerezleri tarayıcı ayarlarınızdan kontrol edebilirsiniz.

4. Güvenlik

Verilerinizi korumak için makul önlemler alıyoruz, ancak hiçbir internet aktarımı %100 güvenli değildir.
